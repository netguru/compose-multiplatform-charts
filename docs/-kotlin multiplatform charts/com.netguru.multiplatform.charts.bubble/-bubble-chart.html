BubbleChart

Represents the data in a form of bubbles that can overlap and are sized based on data value ratios. The position of bubbles is not stable and will likely change every time it is loaded!

Parameters

bubbles

Data to represent

animation

Animation to use for displaying the data

distanceBetweenCircles

Distance between circles in pixels. If negative, bubbles will overlap.

bubbleLabel

Composable to show in the middle of the bubble.
