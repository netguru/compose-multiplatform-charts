Donut shaped 2D pie chart, portraying the values based on the ratios between them.

This chart uses no labels to describe the data. In case labels are needed, use PieChartWithLegend

Parameters

data

Data to show

animation

Animation to use. ChartAnimation.Sequenced is currently not supported and will throw an kotlin.UnsupportedOperationException if used.

Throws

kotlin.UnsupportedOperationException

when ChartAnimation.Sequenced is used
