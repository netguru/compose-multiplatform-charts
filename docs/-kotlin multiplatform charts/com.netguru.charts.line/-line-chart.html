Classic line chart with some shade below the line in the same color (albeit with a lot of transparency) as the line and floating balloon on touch/click to show values for that particular x-axis value.

Color, shape and whether the line is dashed for each of the lines is specified in the LegendItemData class. Even though that this class is used, this particular composable does not show the legend. For this, LineChartWithLegend must be used.
